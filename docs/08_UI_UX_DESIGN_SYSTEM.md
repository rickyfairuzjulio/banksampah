# 1. Design Philosophy

SiSampah bukan sekadar website Bank Sampah.

SiSampah adalah platform digital modern yang menggabungkan:

- Enterprise Dashboard
- Fintech Experience
- Smart City
- Artificial Intelligence
- Environmental Technology

Vercel

GitHub

Apple Dashboard

# 2. Design Principles

Semua halaman harus mengikuti prinsip berikut.

## Simple

Tidak ada elemen yang tidak memiliki fungsi.

## Fast

Semua informasi penting terlihat dalam 3 detik pertama.

## Consistent

Seluruh komponen menggunakan Design Token yang sama.

## Responsive

Desktop

Laptop

Tablet

Mobile

TV (Future)

## Accessible

Mengikuti WCAG 2.2 AA

# 43. Design Token

Semua warna, spacing, radius, shadow, typography, animation, border, dan icon harus menggunakan Design Token.

Tidak diperbolehkan menggunakan nilai hardcoded pada komponen.

---

# 44. UI Quality Standard

Seluruh halaman harus memenuhi:

✓ Lighthouse ≥ 95

✓ WCAG AA

✓ CLS < 0.1

✓ LCP < 2.5 detik

✓ INP < 200 ms

✓ Responsif di seluruh breakpoint

✓ Konsisten dengan Design System

✓ Mendukung Light & Dark Mode

---

# 45. Final Design Goal

Desain SiSampah harus memberikan kesan:

- Modern seperti Linear
- Bersih seperti Apple
- Informatif seperti Google Analytics
- Cepat seperti Vercel
- Profesional seperti Stripe
- Mudah digunakan oleh masyarakat umum
- Layak dipresentasikan pada kompetisi nasional maupun internasional

Setiap komponen yang dibuat harus mengutamakan pengalaman pengguna, aksesibilitas, performa, serta konsistensi visual agar SiSampah memiliki identitas yang kuat sebagai platform pengelolaan sampah digital berbasis teknologi modern.
